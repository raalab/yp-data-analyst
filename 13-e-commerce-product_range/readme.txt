Все папки с рабочими листами Jupyter Notebook содержат необходимый датасет, 
дополнительные настройки путей - не требуются.

Основной проект 
"Анализ товарного ассортимента интернет-магазина "Пока все еще тут"
(корневой католог архива):
product_range-v1.ipynb - файл jupyter основного проекта
ecommerce_dataset.csv - датасет основного проекта
requirements.txt - файл с перечнем необходимых библиотек

Дополнительный поект по SQL:
SQL\  
SQL-v3.ipynb - файл jupyter доп. проекта по SQL
params.txt - параметры подключения к БД
description_data.txt - описание поле БД

Дополнительный поект по A/B тестированию:
Dop_ABtest\  
ABtest-v2.ipynb - файл jupyter доп. проекта по SQL
ab_log.csv - датасет с логами
